Scientific computing involves working with large, distributed datasets. This requires effective use of network capacity and the means to optimize the transfer of actual data between remote compute and storage nodes. Phoebus is an architecture that increases network throughput by intelligently segmenting and routing across heterogeneous paths. Phoebus can make decisions about network optimization based on network information provided by the perfSONAR framework.

The REDDnet (Research and Education Data Depot network) project is designed to provide a large distributed storage facility for data intensive collaboration. For this challenge, we will use REDDnet resources to provide working storage for a terabyte-scale network measurement data set. The integration of Phoebus will facilitate the massive distribution of data from REDDnet and will use a model similar to that planned for Large Hadron Collider data, namely a hierarchy of dissemination and analysis.
